## Why GitHub Actions?

After multiple attempts to build locally on your Mac with the NDK, we encountered architecture/linking issues. GitHub Actions avoids these problems by:
- Building on pre-configured Linux servers
- Proper Android toolchain setup
- Standardized build environment
- Automatic artifact upload to release
